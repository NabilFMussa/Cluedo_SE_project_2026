💡 Best ideas for a better local multiplayer experience
If the focus is same-screen / pass-and-play, these would help most:

1. Pass-to-next-player screen
Before each turn:

hide all private info
show Pass to [player name]
press a key to continue
This is probably the most important local multiplayer feature.

2. Private hand / notes reveal
Let the current player:

hold a key or click a button to reveal their hand
hide it again before passing the device on
3. Clickable setup menu
Right now setup is keyboard-based.
A mouse-friendly version with buttons would feel much better for casual players.

4. Clear turn prompts
Add bigger messages like:

Roll the dice
Choose where to move
Make a suggestion or end turn
This makes pass-and-play smoother.

5. Suggestion result popups
When someone disproves a suggestion, show it in a cleaner popup instead of only the event log.

6. Detective notes improvements
For local multiplayer, a nicer notes screen would help a lot:

click to mark cards
✓, ?, X
separate suspect / weapon / room tabs
7. Restart / new game shortcuts
Useful extras:

Play again
Back to menu
maybe same players again
8. Visual polish
A few small upgrades would improve the feel:

clearer door highlights
cleaner fonts
larger buttons
better spacing in side panel
less clutter on screen
